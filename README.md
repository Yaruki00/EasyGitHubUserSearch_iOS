# なにこれ

だいたいこれができればアプリ作れるかなーという基準にしたい。
![demo](https://github.com/Yaruki00/EasyGitHubUserSearch_iOS/blob/master/demo.gif)

- API通信
- JSONのパース
- ディレクトリ構成
- ファイル/関数分割のやり方
- レイアウトファイルの使い方
- AutoLayouの使い方
- UITableViewの使い方
